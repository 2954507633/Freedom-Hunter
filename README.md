# Freedom-Hunter

Monster Hunter like game, made with [Godot 2.1][godot].

## Controls

Key                     | Action
------------------------|-----------------
<kbd>w/a/s/d</kbd>      | Move
<kbd>Shift</kbd>        | sprint
<kbd>Space</kbd>        | jump
<kbd>Left click</kbd>   | primary attack
<kbd>Right click</kbd>  | secondary attack
<kbd>Middle click</kbd> | center camera
